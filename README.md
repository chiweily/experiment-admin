# 实验管理系统 
## 简介
本项目基于 react、next.js、antd 技术栈实现，目前系统主要包含三个子模块，分别是：实验管理、实验选择、用户管理。后续将会继续完善各个子模块以及添加服务端模块代码（待学习）。  
本项目的 Apifox 的接口可以访问[这个地址](https://apifox.com/apidoc/shared-69240c11-2661-4e6d-abeb-14725cb93e9f)
